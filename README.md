# Cub3d

## Description

Cub3D is a raycasting-based 3D maze game inspired by Wolfenstein 3D, built in C using the MiniLibX graphics library.

- - - - - - -

## Instructions

### Compilation :
```bash
make 
```

### Parameters : 

- `map.cub` : Run a .cub file as a map

### Usage :

```bash
./cub3d maps/map.cub
```

- Move with WASD.
- Look with left and right arrow keys.
